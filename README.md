# SaveLobby

*"Una comunidad por gamers, para gamers, y sobre gaming."*

## Sobre el Proyecto
**SaveLobby** es una plataforma web y PWA (Progressive Web App) independiente, diseñada 
como el punto de encuentro definitivo para la comunidad de videojuegos. A diferencia de 
foros tradicionales, SaveLobby cuenta con una base de datos dinámica de juegos, un 
sistema robusto de etiquetas (MODS, GUIA, DUDAS, AYUDA, y más que añadiremos), soporte 
multimedia y una economía interna orientada a recompensar a los usuarios y creadores de 
contenido.

## Características Principales
- **Foros Dinámicos:** Selección y adición automatizada de videojuegos a la base de 
datos al crear una publicación.
- **Identidad Única:** Nombres de usuario irrepetibles (cambios de nombre habilitados 
únicamente mediante micropagos).
- **Economía y Recompensas (P2P):** Sistema integrado para donar dinero real a otros 
usuarios por sus aportes, guías o resoluciones de dudas.
- **Secciones Privadas:** Zonas exclusivas o *paywalls* gestionados por creadores de 
contenido.
- **Accesibilidad:** Soporte nativo PWA para instalación directa en dispositivos móviles 
y PC.

# SaveLobby

*"A community by gamers, for gamers, and about gaming."*

## About the Project
**SaveLobby** is an independent web platform and PWA (Progressive Web App), designed as 
the ultimate meeting point for the gaming community. Unlike traditional forums, 
SaveLobby features a dynamic game database, a robust tagging system (MODS, GUIDE, 
QUESTIONS, HELP,and more features), multimedia support, and an internal economy focused 
on rewarding users and content creators.

## Main Features
- **Dynamic Forums:** Automated selection and addition of games to the database when 
creating a post.

- **Unique Identity:** Unique usernames (name changes enabled only through 
microtransactions).

- **Economy and Rewards (P2P):** Integrated system for donating real money to other 
users for their contributions, guides, or answers to questions.

- **Private Sections:** Exclusive areas or paywalls managed by content creators.

- **Accessibility:** Native PWA support for direct installation on mobile devices and 
PCs.
